                     Social Media plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with Social Media 2.2.0

----------------------------------------------------------------------------
INSTALLATION:

  1. Copy plugin files to amember/plugins/protect/socialmedia/ folder

  2. Enable plugin at aMember CP -> Setup/Configuration -> Plugins

  3. Configure plugin at aMember CP -> Setup/Configuration -> Social Media

  4. Adjust product settings at aMember CP -> Manage Products -> Edit: Social Media Group/Channel
----------------------------------------------------------------------------
