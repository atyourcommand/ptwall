                     XCart plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with XCart Gold v4.1.3

----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to amember/plugins/protect/xcart/ folder
  2. Enable plugin at aMember CP -> Setup -> Plugins
  3. Configure plugin at aMember CP -> Setup -> XCart
  4. Adjust product settings at aMember CP -> Edit Products
----------------------------------------------------------------------------
