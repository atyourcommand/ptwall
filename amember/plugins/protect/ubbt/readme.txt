1. Enable plugin in aMember Admin CP/Setup, then go to plugin settings
page and configure it.

2. If you want to setup different levels of access to forum for 
different products, just add new user groups into your UBBT database.

3. Login into aMember Control Panel. Add Products and set correspoding 
UBBT Group for every product. Members will get this level of access
in UBBT as subscription bought. When subscription expires it will be
reset to default level. 

4. After you set it, click on Rebuild Db link in Admin Panel. It will 
cause database syncronisation (aMember Pro ----> UBBT, but
never UBBT ----> aMember Pro)

5. Enjoy!

NOTES:
If user do signup with the same login AND email as in UBBT, then 
it is allowed to do signup. If user do login with existing (in UBBT)
login and ANOTHER email, it will get message that username is already
taken. 

If you have existing UBBT members database contact CGI-Central
for free advice about import/integration.
