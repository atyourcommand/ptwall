                     Typo3 plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with Typo v.3.6.0RC2

----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to amember/plugins/protect/typo3/ folder
  2. Enable plugin at aMember CP -> Setup -> Plugins
  3. Configure plugin at aMember CP -> Setup -> Typo3
  4. Adjust product settings at aMember CP -> Edit Products
----------------------------------------------------------------------------
