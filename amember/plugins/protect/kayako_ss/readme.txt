                     Kayako SupportSuite plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with Kayako SupportSuite 3.30.01

----------------------------------------------------------------------------
INSTALLATION:

  1. Copy plugin files to amember/plugins/protect/kayako_ss/ folder

  2. Enable plugin at aMember CP -> Setup/Configuration -> Plugins

  3. Configure plugin at aMember CP -> Setup/Configuration -> Kayako SupportSuite

  4. Adjust product settings at aMember CP -> Manage Products -> Edit: Kayako User/Staff access
----------------------------------------------------------------------------
