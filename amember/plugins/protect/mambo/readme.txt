                     Mambo module installation

                                   
1. Enable plugin in aMember Admin CP -> Setup -> Plugins, 

2. Go to aMember CP -> Setup -> Mambo and configure it.

3. Login into aMember Control Panel -> Edit Products and set correspoding 
Mambo access group for every your product. 

4. Enjoy!

There is also module mod_amemberl.zip for Mambo in 
 amember/plugins/protect/mambo/ folder.
