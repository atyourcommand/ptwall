                     iGiveTest plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with iGiveTest 1.9.7

----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to amember/plugins/protect/igivetest/ folder
  2. Enable plugin at aMember CP -> Setup -> Plugins
  3. Configure plugin at aMember CP -> Setup -> iGiveTest
  4. Adjust product settings at aMember CP -> Edit Products
----------------------------------------------------------------------------
