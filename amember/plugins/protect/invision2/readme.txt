                     Invision2.2 plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with Invision Board  v2.2.1 
  This version of the plugin requires aMember 3.0.8PRO
  If you have older version, please upgrade
----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to amember/plugins/protect/invision2/ folder
  2. Enable plugin at aMember CP -> Setup -> Plugins
  3. Configure plugin at aMember CP -> Setup -> invision2
  4. Adjust product settings at aMember CP -> Edit Products
----------------------------------------------------------------------------
