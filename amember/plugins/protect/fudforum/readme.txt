                fudForum PLUGIN README

INSTALLATION
------------
   * extract archive file to amember/plugins/protect folder
   * go to aMember CP => Setup/Configuration => Plugins
   * enable "fudforum" plugin
   * go to aMember CP => Setup/Configuration => fudForum
   * set database name and prefix (prefix is fud23_ by default)
   * go to aMember CP => Edit Products and assign FudForum integration
     for every product
   * login into fudForum configuration panel and set cookie name 
     to "fud_session"
   * it is all. enjoy!
