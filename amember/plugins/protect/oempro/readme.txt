<b>oemPro Plugin</b>         

oemPro Plugin is designed to provide access to oemPro Mail Lists
for users in aMember Pro.

<b>Installation</b>
1 - Enable plugin at aMember CP -> Setup/Configuration -> Plugins

2 - Configure plugin at 
aMember CP -> Setup/Configuration -> oemPro

<b>Setup</b>
Set Setup/Configuration -> Global -> Plugins Require Unique Email

The plugin adds configuration fields for database name and 
database table prefix.
Options about deleting users from oemPro.
And aMember fields mapping.

<b><u>Database name and table prefix</u></b>:
Format is &quot;database_name.table_prefix&quot;. 

<b><u>Remove users</u></b>:
Choose aMember Pro action when user removed from aMember.

<b><u>Fields Mapping</u></b>:
Map aMember additional fields to oemPro custom fields.
