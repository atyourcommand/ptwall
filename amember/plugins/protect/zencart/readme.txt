<b>Zen-Cart plugin</b>


----------------------------------------------------------------------------
REQUIREMENTS:

Zen-Cart (tested with 1.3.8 version)
http://www.zen-cart.com/

----------------------------------------------------------------------------
INSTALLATION:

1. Extract and upload plugin to amember/plugins/protect/zencart/ folder

2. Enable plugin in aMember Admin CP/Setup, then go to plugin settings
   page and configure it.

STRONGLY RECOMENDED :
----------------------------------------------------------------------------
For aMember
  Enable : CP -> Setup/Configuration -> Global -> Require Unique Email
  
  Disable e-mail in : CP -> Setup/Configuration -> Advanced -> User can change the following fields

For Zen-Cart
  Session must be store into Database(not in files).
