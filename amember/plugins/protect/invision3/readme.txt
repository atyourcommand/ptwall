                     IP.Board 3 plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with IP.Board v3.0.1 
  This version of the plugin requires aMember 3.1.8PRO
  If you have older version, please upgrade
----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to /amember/plugins/protect/invision3/ folder
  2. Enable plugin at aMember CP -> Setup/Configuration -> Plugins
  3. Configure plugin at aMember CP -> Setup/Configuration -> IP.Board 3
  4. Adjust product settings at aMember CP -> Manage Products -> Edit
----------------------------------------------------------------------------
