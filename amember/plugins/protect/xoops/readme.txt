                     XOOPS Plugin Notes

----------------------------------------------------------------------------
REQUIREMENTS:

XOOPS 2.0.6 or above installed

----------------------------------------------------------------------------
INSTALLATION:

1. Enable plugin in aMember Admin CP/Setup, then go to plugin settings
page and configure it.

2. Edit your products via aMember Pro Control Panel and set XOOPS 
   Access value, also set Product URL to point to XOOPS installation.

3. Check your installation by doing new signup. When signup is completed,
   new member must appear in XOOPS with the same username and password.

----------------------------------------------------------------------------

NOTES:
Plugin tested with XOOPS 2.0.6, 2.0.16 and 2.2.5 rc2
