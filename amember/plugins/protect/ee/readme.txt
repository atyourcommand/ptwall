                     Expression Engine plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with Expression Engine v.1.4 and v1.5.2

  It is necessary to choose SHA1 encoding for passwords (default)
  in ExpressionEngine installation wizard.

----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to amember/plugins/protect/ee/ folder
  2. Copy core.sha1.php from your EE system/core/ to amember/plugins/protect/ee/
  3. Enable plugin at aMember CP -> Setup -> Plugins
  4. Configure plugin at aMember CP -> Setup -> EE
  5. Adjust product settings at aMember CP -> Edit Products
  6. Click aMember CP -> Rebuild Db to transfer existing members to EE
----------------------------------------------------------------------------
