                     ClipShare plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with ClipShare 2.5

----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to amember/plugins/protect/clipshare/ folder
  2. Enable plugin at aMember CP -> Setup -> Plugins
  3. Configure plugin at aMember CP -> Setup -> ClipShare
  4. Adjust product settings at aMember CP -> Edit Products
----------------------------------------------------------------------------
