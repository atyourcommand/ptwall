                
                phpWebThings PLUGIN README

INSTALLATION
------------
   * extract attached file to amember/plugins/protect folder
   * go to aMember CP => Setup/Configuration => Plugins
   * enable "phpwebthings" plugin
   * go to aMember CP => Setup/Configuration => phpWebThings
   * set database name and prefix (prefix is wt_ by default)
   * set URL (you can find this url value in the file
     phpwebthings/core/config_vars.php, line like
     $config_var["core"]["url"]["default"] = "http://localhost/";
     http://localhost/ is value you need to set in aMember
   * go to aMember CP => Edit Products and set phpWebThings integration
     for every product
   * it is all. enjoy!
