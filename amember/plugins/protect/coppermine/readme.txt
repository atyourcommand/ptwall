                     Coppermine Photo Gallery plugin notes

----------------------------------------------------------------------------
REQUIREMENTS:
  Plugin has been tested with Coppermine Photo Gallery 1.4.10

----------------------------------------------------------------------------
INSTALLATION:
  1. Copy plugin files to amember/plugins/protect/ñoppermine/ folder
  2. Enable plugin at aMember CP -> Setup -> Plugins
  3. Configure plugin at aMember CP -> Setup -> Coppermine
  4. Adjust product settings at aMember CP -> Edit Products
----------------------------------------------------------------------------
