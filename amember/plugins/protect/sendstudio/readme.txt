                     SendStudio Plugin

This plugin has been tested with SendStudio 1.3.3

SendStudio Plugin installation
-----------------------------------------
0. Copy sendstudio.inc.php, config.inc.php and readme.txt to amember/plugins/protect/sendstudio 
1. Enable 'sendstudio' plugin in aMember Admin CP -> Setup -> Plugins from Protect Plugins list
2. Go to aMember CP -> Setup -> SendStudio and configure it.
3. Go to aMember CP -> Manage Products -> edit and set correspoding 
   "SendStudio access" lists. 
4. To transfer your active members to SendStudio, click aMember CP -> Rebuild Db
