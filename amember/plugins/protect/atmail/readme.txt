@Mail integration plugin

Introduction.
This plugin allows to manage (sell) @Mail accounts from aMember
and to sell SMS packages as well.

Installation:

1. Upload plugin files to amember/plugins/protect/atmail/ folder.
2. Enable plugin at aMember CP -> Setup -> Plugins
3. Configure plugin at aMember CP -> Setup -> @Mail
4. Visit aMember CP -> Edit Products and adjust products settings
 related with atmail.
